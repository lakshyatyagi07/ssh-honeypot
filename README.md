# 🛡️ Risk-Aware SSH Honeypot

![Python](https://img.shields.io/badge/Python-3.x-blue)
![Cybersecurity](https://img.shields.io/badge/Cybersecurity-Honeypot-red)
![License](https://img.shields.io/badge/License-MIT-green)

A Python-based medium interaction SSH honeypot designed to simulate an SSH server, monitor attacker behavior, log malicious activities, track executed commands, and visualize attack data through a web dashboard.

---

# 📌 Features

✅ SSH Honeypot Simulation  
✅ Attacker Activity Tracking  
✅ Command Monitoring & Logging  
✅ Risk-Aware Logging System  
✅ Web Dashboard Interface  
✅ Fake File System Simulation  
✅ Deception & Realism Modules  
✅ Real-Time Attack Analysis  

---

- Python

# 📂 Project Structure

```bash
project/
│
├── main.py                # Main honeypot server
├── attacker.py            # Attacker activity handling
├── commands.py            # Command processing
├── dashboard.py           # Web dashboard backend
├── deception.py           # Deception mechanisms
├── filesystem.py          # Fake filesystem simulation
├── realism.py             # Realistic environment behavior
├── tracker.py             # Activity tracking
├── honeypot_log.json      # Attack logs

```

# ⚙️ Installation

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/yourusername/risk-aware-ssh-honeypot.git
```

## 2️⃣ Navigate to Project Directory

```bash
cd risk-aware-ssh-honeypot
```

## 3️⃣ Install Required Dependencies

```bash
pip install flask
```
